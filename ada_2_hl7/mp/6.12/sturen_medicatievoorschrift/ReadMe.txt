The ada_instance folder contains the contents in a 'simple dataset xml format'. It is based on ADA format, more information about ADA can be found here:
https://www.art-decor.org/mediawiki/index.php/ADA_Walkthrough. It is not necessary to read all this to use the example xml's. 
The xsd of the ada format is included in the folder "ada_schemas".

The folder 'ada_instance' contains sample ada instance file(s). They are in no way functionally valid (i.e. the situation can probably not be validated in real life). The folder hl7_instance contains the result of the conversion for these sample files. The folder vzvz_instance provides the same data with example AORTA-wrappers as are used on LSP.
