This publication has some conversion xslt's which convert a MP 9.0.6 medicatievoorschrift to a generic 'ADA'-format (hl7_2_ada\mp\9.0.6\sturen_medicatievoorschrift).
This publication also has some conversion xslt's which convert a generic 'ADA'-format to a MP 6.12.8 vooraankondiging (ada_2_hl7\mp\6.12.8\sturen_medicatievoorschrift).

Combining the two conversions above gives you a 9.0.6 to 6.12.8 conversion.

Further documentation can be found in:

hl7_2_ada\mp\9.0.6\sturen_medicatievoorschrift\Readme.txt
ada_2_hl7\mp\6.12.8\sturen_medicatievoorschrift\Readme.txt
